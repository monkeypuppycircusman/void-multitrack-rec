VOID DAW — single file, runs offline. Here's what's packed in:
Multitrack

4 default tracks, add unlimited more with T or the button
Drag .wav/.mp3/.ogg from the file panel onto any track lane
Drag clips around between tracks, resize from left/right edges
Cut tool slices at cursor, erase tool wipes a clip
Snap-to-grid toggle (G) with quarter-second grid
Zoom with +/- or Ctrl+scroll

Editing

Right-click any clip → split, copy, paste, duplicate, rename, delete
Double-click a clip → opens it in the Waveform Editor below
In the wave editor: select regions by dragging, then cut / fade in / fade out / normalize / reverse
Drag the resize handle to make the wave editor taller

Recording

Go to the MIC tab → Init Mic, then ARM a track (hit R on the track header)
Hit ⏺ REC — it records from your mic in real-time and drops the clip on the armed track

Transport

Space = play/pause, R = record, S = stop/split, Home = rewind
Loop toggle, live VU meters, BPM display, time counter

Save

💾 SAVE exports a .json project file with all your track/clip layout

Just open the file in any modern browser — no install, no server, no internet needed.
